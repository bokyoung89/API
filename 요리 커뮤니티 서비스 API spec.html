Article API

게시글 리스트 조회

설명

조회한 데이터를 20건씩 JSON 형식으로 반환합니다.

요청 url

$ http GET 'http://localhost:8080/api/articles'

프로토콜

HTTPS

HTTP 메서드

GET

응답

응답에 성공하면 결괏값을 JSON 형식으로 반환합니다.

설명

요청 url

$ http GET 'http://localhost:8080/api/articles/1'

articleId
